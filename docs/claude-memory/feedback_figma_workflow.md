사용자는 Figma Make MCP를 통해 디자인 → 코드 변환 워크플로우를 선호함.
Figma Make로 페이지를 생성시키고, 그 결과를 Claude가 받아서 코드로 구현하는 방식.

**Why:** 디자인 품질과 일관성을 위해 Figma 기반 개발 프로세스를 원함.
**How to apply:** UI 작업 시 Figma 디자인 링크/선택을 먼저 받고 코드로 변환. Figma MCP 연결 상태 확인 필요.
